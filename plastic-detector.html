Step 1 — paste your model URLs

Upload your 3 files (model.json, weights.bin, metadata.json) somewhere accessible — GitHub raw URLs work great. Then paste the base URL below (the folder path, not the file itself).

Example: https://raw.githubusercontent.com/yourname/yourrepo/main/model/

How to host your files for free

  1. Go to github.com and sign in (or create a free account)
  2. Click + → New repository, name it anything, make it Public
  3. Click Add file → Upload files and upload all 3 files
  4. Open each file, click Raw, and copy the URL up to the last /
  5. Paste that base URL above
